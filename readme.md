# Getting Started:  
- **createdb gulfport_votes** for server to be able to be run with database
While in server directory:  

```bash
python3 seed.py
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
flask run
```

- this backend is intended to be used with the following frontend repo: [Gulfport_Frontend](https://github.com/la-flor/Gulfport_Frontend)
